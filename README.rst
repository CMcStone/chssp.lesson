============
chssp.lesson
============

Custom lesson content type. Features lesson metatdata and a link to lesson cloud storage

Features
--------

- works with collective.taxonomy


Installation
------------

Install chssp.lesson by adding it to your buildout::

    [buildout]

    ...

    eggs =
        chssp.lesson


and then running ``bin/buildout``
